# Task Tracker Application

A full-stack Task Tracker application that helps users create, manage, update, and track their daily tasks. The application provides an easy-to-use interface with task management features and a responsive design.

## 🚀 Features

### User Features
- User registration and login authentication
- Create new tasks
- View all tasks
- Update task details
- Delete tasks
- Mark tasks as completed/pending
- Manage tasks efficiently
- Responsive user interface

### Application Features
- REST API integration
- Database storage
- Secure authentication
- Clean and responsive UI
- Real-time task status updates
